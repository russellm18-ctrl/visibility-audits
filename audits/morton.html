Visual Authority Audit

Morton

https://www.msjaia.com/
Visual Authority Grade
D+

Architecture firm website relies entirely on outdated hero carousel and static featured project thumbnails with zero modern visual storytelling—no before/afters, no project video testimonials, no interactive 3D walkthroughs, no client case study photography sequences.

Asset Quality Gaps

Hero carousel images appear generic (974x385px low-resolution placeholders); featured project thumbnails (160x108px) severely undersized for architectural portfolio; no high-resolution detail shots of craftsmanship, materials, or interiors; missing contextual photography showing design problem-solving; no photography metadata or EXIF standards visible; carousel uses fade transitions instead of intentional narrative pacing between projects.

Mission Control Gaps

Zero project management infrastructure for asset versioning—no image alt-text strategy beyond generic 'Slide' labels; no structured portfolio taxonomy; carousel autoplay (6000ms timeout) treats all projects as equal weight; no dynamic asset serving based on device/viewport; no lazy-loading optimization; featured carousel has hard-coded 5-item visible limit creating artificial scarcity; no analytics event tracking on project image clicks or carousel interactions.

Analytics & Speed Gaps

No image optimization—raw JPG files loaded at full resolution (974x385px hero); multiple CSS/JS framework loads (jQuery 1.7.2, Cufon font rendering, jCarousel) bloat initial paint; timthumb.php image resize middleware creates runtime overhead; no WebP format variants; LiteSpeed cache headers present but image delivery unoptimized; no viewport-specific image serving; carousel JavaScript initializes on page load regardless of user engagement.

The Revenue Leak

Portfolio fails to convert architectural inquiry traffic—low-resolution project images undermine perceived quality/caliber of firm; no visual proof of LEED/AIA awards mentioned in copy (awards mentioned but not visually documented); carousel format treats residential, remodel, and civic projects identically, losing opportunity to upsell premium service tiers; missing client testimonial photography; no before/after gallery storytelling that justifies premium design fees; featured projects lack context (budget, timeline, client transformation narrative).

The Photo-First Solution

Rebuild portfolio as photo-first asset management system: (1) Capture structured photography sets per project—exterior approach, materials detail, interior spaces, user occupation, final completion shots; (2) Implement tiered image delivery—hero carousel at 1920x1080 with WebP+JPG fallback, project detail at 800x600 thumbnail + 3000x2000 lightbox; (3) Create project template: hi-res before/after slider gallery, 3-5 sequential detail photographs with captions explaining design decisions, client testimonial video thumbnail; (4) Tag all images with schema markup (schema.org/CreativeWork + photographer credit); (5) Replace auto-cycling carousel with user-driven gallery revealing one project deeply per interaction; (6) Establish monthly photography audit cycle—quarterly reshoot of completed projects with drone/staging photography to refresh portfolio authority signals.
